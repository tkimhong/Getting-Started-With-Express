# José Ramírez Fan Page - Cleveland Guardians Tribute

Express.js web app featuring José Ramírez and the Cleveland Guardians.

## Live demo

[José Ramírez Cleveland Guardians](https://tkimhong-inf653-hw2.onrender.com)

## Features

- **Home**: José Ramírez career stats, highlights, and JRAM Facts Every Friday
- **About**: "DOWN GOES ANDERSON" and Tim Anderson post-punch stats
- **Contact**: Steven Kwan "Contact King" with awesome contact skills on display
- **404 Page**: Jonathan Villar slide "Second Base Not Found"

## Run

```bash
npm install
node index.js
```
Visit `http://localhost:3000`
